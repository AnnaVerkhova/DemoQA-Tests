DemoQA-Tests — учебный тестовый проект по автоматизации API и UI тестирования на основе сайта DemoQA
.

Проект создан в рамках практического задания для отработки навыков:

- построения архитектуры автотестов

- работы с REST API

- написания UI-тестов

- использования TestNG, Maven и Allure

- построения базовой структуры test framework

Проект носит учебный характер и предназначен для демонстрации навыков автоматизации тестирования.

**Технологический стек:**

- Java 

- Maven

- TestNG

- RestAssured (API тестирование)

- Selenium WebDriver (UI тестирование)

- Lombok

- Allure Report

- Owner (для работы с конфигурациями)
